# ✅ Todo App with Redux Toolkit

A simple and efficient **Todo App** built using **React** and **Redux Toolkit**, featuring full **CRUD operations** — add, edit, delete — and a **fully responsive design** for all screen sizes (mobile, tablet, desktop).

---

## ✨ Features

- 🆕 **Add Todo** – Quickly add new tasks
- 📝 **Edit Todo** – Update tasks inline with ease
- 🗑️ **Delete Todo** – Remove completed or unwanted tasks
- ⚙️ **State Management** – Handled via Redux Toolkit for clean and scalable architecture
- 📱 **Responsive Design** – Seamless experience across all devices
- 🔄 **Real-time UI updates** on every action

---

## 🧠 Tech Stack

- **React** – Functional Components with Hooks
- **Redux Toolkit** – Slices, store, and reducers
- **HTML & CSS** – Clean and responsive UI
- **JavaScript (ES6+)**
